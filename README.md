# Giovani Machado

Personal site for a short introduction, a few public opinions, selected projects, and a downloadable CV.

The goal is to feel closer to a personal note than a traditional resume: plain text, minimal interface, and enough personality to show how I think about software, product work, AI, trade-offs, and building useful systems.

The site is intentionally small, bilingual, and static. It should be easy to read, easy to maintain, and easy to deploy anywhere that serves simple web assets.
